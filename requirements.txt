# =====================================================================
# Requirements -- MAIN environment

# This is the env that runs video_subtitle_translate.py.
# It loads: Gemma 4 (refiner), stable-ts (forced alignment), SaT
# (sentence segmentation), audio-separator (vocal isolation), and the
# Tencent Hunyuan-MT2 translator.
#
# It does NOT load Qwen3-ASR. Qwen runs in a SEPARATE env that serves an
# OpenAI-compatible HTTP API; this script just sends HTTP requests to it.
# See QWEN_ASR_SERVER_SETUP.md and requirements-qwen.txt.
#
# SYSTEM DEPENDENCIES (not pip-installable):
#   ffmpeg  (must be on PATH and provide both `ffmpeg` and `ffprobe`)
#     Linux:   sudo apt install ffmpeg
#     macOS:   brew install ffmpeg
#     Windows: choco install ffmpeg
#
# GPU + CUDA NOTE:
#   1. Install torch matched to your CUDA toolkit FIRST from
#      https://pytorch.org/get-started/  (e.g. cu121/cu124 wheels).
#   2. THEN install flash-attention (recommended; the script uses it
#      automatically when present):
#        pip install -U flash-attn --no-build-isolation
#      flash-attn 2 needs Ampere+ (RTX 30/40, A100, H100) and torch loaded
#      in bf16 or fp16 (which this script does).
#   3. THEN  pip install -r requirements.txt
#
# Override the attention backend with USE_FLASH_ATTN=1 (force flash-attn, raise
# if unavailable) or USE_FLASH_ATTN=0 (force sdpa). Default is 'auto'.
